Watson Babel Fish

Watson Babel Fish is an app powered by IBM Watson that translates speech from one language to another!

Select a language on the left to speak, and a language on the right you want to translate that speech in to. Then, hit the microphone to start recording! When done, hit the microphone again, and Watson will go to work. After everything has been translated, it'll speak the result back to you!

Watson Babel Fish is powered by Watson's Speech to Text, Language Translation, and Text to Speech APIs. It's all open source, so go check out the source code.

Watson Babel Fish requires getUserMedia and the Web Audio API to work, so be sure you're in a browser that supports them to use it!
